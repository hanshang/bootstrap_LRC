# bootstrap_LRC
Bootstrap methods for long-run covariance function of stationary functional time series

FAR_boot_longrun_covariance.R: functional autoregressive of order 1 bootstrap method for bootstrapping long-run covariance
SST.txt: Monthly sea surface temperature data
SST_lrc_ex.R: sample long-run covariance function of the sea surface temperature data
essential_functions.R: essential functions with various data generating processes
save_function.R: saving function for R graphics
sieve_boot_longrun_covariance.R: sieve bootstrap method for bootstrapping long-run covariance
sieve_bootstrap.R: sieve bootstrap method for generating bootstrap samples that mimic temporal dependence of the original data
